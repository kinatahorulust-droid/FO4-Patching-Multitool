This tool is designed for use with SAKR Redux. It may also work with the old SAKR version, but you will need the base Complex Sorter settings from the Redux version.

Instructions:

  1. Download this FO4Edit script and copy it to FO4Edit\Edit Scripts.
  2. Run FO4Edit with your target clothing plugin, select it, right-click, and choose Apply Script.
  3. Type ARMO_Export_For_SAKR in search, run the script, and save the result.
  4. Upload the file to this site and edit keywords. You can also upload an existing RobCo Patch for your clothing, and the site will automatically assign all keywords.
  5. To get the result, click Generate INI and Export. Move the output file to Data\Complex Sorter\Plugins.
  6. Run the sorter as usual. If you have never used Complex Sorter, read all instructions from 4estGimp.
